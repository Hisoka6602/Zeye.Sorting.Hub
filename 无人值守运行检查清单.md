# 无人值守运行检查清单

> 用途：面向 365 天无人值守目标的日常巡检清单  
> 关联文档：`生产运行Runbook.md`、`长期运行优化与热更新支持清单.md`、`业务接入前底座验收清单.md`

---

## 一、每日检查

- [ ] `/health/live`、`/health/ready` 正常
- [ ] 慢查询画像无异常暴涨
- [ ] 批量写入队列无持续积压
- [ ] Outbox / Inbox 健康检查无持续堆积
- [ ] NLog 错误日志无同类高频风暴

## 二、每周检查

- [ ] 分表巡检、预建计划、索引检查均正常
- [ ] 数据保留治理与归档任务按周期完成
- [ ] 只读副本预留与报表预算守卫仍处于可用状态
- [ ] 幂等冲突与重复写入无异常升高

## 三、每月检查

- [ ] 对照 `性能基线报告.md` 复核 P50 / P95 / P99 与错误率
- [ ] 检查备份最新文件、恢复 Runbook 路径与健康检查状态
- [ ] 检查日志目录、备份目录、归档目录容量
- [ ] 更新月度巡检报告与风险项

## 四、每季度检查

- [ ] 至少新增一份 `drill-records/` 演练记录
- [ ] 按 `备份恢复演练Runbook.md` 完成恢复演练
- [ ] 复核 `stability-gates.yml` 门禁项是否仍与现状一致
- [ ] 复核 `业务接入前底座验收清单.md` 是否仍全部满足

## 五、发生变更后追加检查

- [ ] 新增业务模块前先通过最终底座验收清单
- [ ] 新增文件后同步更新 `README.md`
- [ ] 更新 `检查台账/文件清单基线.txt`
- [ ] 补充对应检查台账与更新记录
